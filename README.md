# CheloniaTrace | Global Testudines Database

**CheloniaTrace** is a professional-grade biological taxonomy platform built with Next.js. It is designed to provide researchers, conservationists, and turtle enthusiasts with a comprehensive, intuitive, and authoritative visual guide to the world's turtles and tortoises (Order Testudines).

## 🚀 Key Features

* **Interactive Taxonomy Tree**: An advanced divergent organizational chart that allows seamless exploration from Order down to individual Species.
* **Scientific Search**: High-precision search engine optimized for Latin names and common names, featuring auto-expansion to highlight search results within the tree.
* **High-Definition Plates**: Integrated PDF rendering engine for viewing high-resolution morphological plates with support for zooming and free-form dragging.
* **Bilingual Interface**: Toggle between English and Chinese with a single click to cater to a global audience.
* **Optimized UX**: Fully responsive design with a custom-engineered horizontal scrolling experience for desktop and mobile devices.

## 📂 Data & Methodology

The taxonomic structure and nomenclature used in this project are strictly aligned with:
* **IUCN/TFTSG**: [Turtles of the World Checklist and Standard Reference](https://iucn-tftsg.org/checklist/).

## 🛠️ Technical Stack

* **Frontend**: Next.js 15+ (App Router)
* **Language**: TypeScript
* **Styling**: Tailwind CSS v4
* **Deployment**: Cloudflare Pages (Server-Side Rendering)
* **Backend API**: FastAPI (High-performance Python framework)

## 🔧 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone [https://github.com/WhereAreMySOCKS/guiji-next.git](https://github.com/WhereAreMySOCKS/guiji-next.git)
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run in development mode**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```
